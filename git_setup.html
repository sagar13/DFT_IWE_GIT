Quick Terminology
 

  • Go over remote/ local repos. Something like "There are two basic kinds of repositories in Git; remote repositories and local repositories."

    • Remote repositories are most generally defined as repositories hosted OFF your local machine. (Usage: set up a remote repo when a team of developers is working on a single project)

    • Local repositories are most generally defined as repositories hosted ON your local machine. (Usage: set up a local repo when a single developer is working on a single project)

  • Note that Git repos should only be set up once, then every other times cloned from there.
